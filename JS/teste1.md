# Teste rápido: Declaração de variáveis
Qual(is) das alternativas a seguir mostram o valor 10 no console?<br>
**A)**
```
const check = true;
if(check){
    let variavel = 10;
}
console.log(variavel);
```
**B)**
```
const check = true;
if(check){
    const variavel = 10;
}
console.log(variavel);
```
**C)**
```
const check = true;
if(check){
    var variavel = 10;
}
console.log(variavel);
```
**D)**
```
const check = true;
if(check){
    window.variavel = 10;
}
console.log(window.variavel);
```
